# Multi Role Emoji

!!! example "Command Usage"
    z/multi emoji rolename, rolename, etc

***Or*** 

!!! example "Command Usage"
    z/multi emoji rolemention, rolemention, etc


### Description

Setup a reaction role that will grant multiple roles on react.
